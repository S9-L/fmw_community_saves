This archive contains a series of save files with special features designed for freely playing in the NG+ sandbox.
 
Included are the following:
- Every character is carried over into NG+ as if they were part of the selected 6 NG+ units group in an NG+ playthrough that has completed Route B.
-- Hourai, Shanghai, Zenki, and Goki are made available by default without needing to bypass NG+ carryover selection logic.
- Points, Items, Character PP, Character Kills, are all reset back to 0 or their defaults, as if the game was set to a 1st playthrough.
-- Ex-Points are set to 10 to ensure that the player will receive the expected WP total by the time they get access to EX10.
- Starting chapter is set to the 2nd chapter of either route, since it's the earliest that a save file can be created. The 1st chapter's clear records are configured for a 1-turn clear on Lunatic with the bonus achieved. This can be changed through the save manager, if desired.
- There are 2 overall version groups for the save files. 1st Group for NG+ freeplay inlcudes all the features above, while the 2nd Group unlocks all PSes regardless of level/event restrictions and makes them available at the start.

Rename the number in the save file name to whatever 3-digit save file slot number that is available.
Place your newly renamed save file in the following folder for Steam and you should be able to see your new save file in-game:
C:\Users\<YOUR_USERNAME>\AppData\Local\fmw_dosd\savedata

Switch saves require CFW on original hardware or an emulated save folder. You should follow whatever guide you used to setup save management on Switch CFW or emulator to properly load these Switch saves.

These save files work on all vanilla/modded versions of Fantasy Maiden Wars - Dream of the Stray Dreamer on both Switch and Steam, up to 1.2.4.x Steam and 1.2.5.x Switch, as of 7-15-2026. Pick the file relevant to your platform, and the save file will work out of the box. For further troubleshooting and/or discussing the game, reach out in the unofficial FMW Discord!
